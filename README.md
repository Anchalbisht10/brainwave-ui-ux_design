🌐 Brainwave UI/UX Website

A modern and responsive AI SaaS landing page built using React.js, Vite, and Tailwind CSS, showcasing advanced UI/UX design principles and smooth interactive animations.

---

🚀 Live Demo
[
👉 [https://brainwave-ui-ux-design.vercel.app]

---

📌 Overview

This project is a visually rich frontend application designed to replicate a modern AI product landing page. It focuses on clean design, smooth animations, and reusable component architecture.

---

⚙️ Tech Stack

- ⚛️ React.js
- ⚡ Vite
- 🎨 Tailwind CSS

---

✨ Features

- 📱 Fully responsive design across devices
- 🎯 Modern UI/UX layout (Hero, Features, Pricing, Roadmap)
- 🎞️ Smooth animations and transitions
- 🧩 Reusable component-based structure
- 🎨 Gradient effects and visually appealing sections

---

📁 Project Structure

src/
│── assets/        # Images, icons, SVGs
│── components/    # UI components
│── constants/     # Static data
│── App.jsx        # Main layout
│── main.jsx       # Entry point
